# Senior Product Designer — Workshop Kit

Everything needed to run the "design like a senior, inside YOUR product" workshop.

## Contents
- **`guide.html`** — the full step-by-step guide (open in a browser). Start here.
- **`setup.sh`** — one-command installer for all dependencies.
- **`skills/`** — bundled skills:
  - `senior-product-designer/` — the conductor skill (the one you build live in the workshop; this is the reference copy)
  - `ui-ux-pro-max/` — the UX correctness backbone

## Quick start
```bash
bash setup.sh          # installs everything
```
Then **restart Claude Code** and open `guide.html` in a browser.

## What gets installed
| Skill / plugin | Source | Role |
|---|---|---|
| frontend-design | official marketplace | taste / delight |
| ui-ux-pro-max | bundled here | UX correctness |
| design-systems | designer-skills marketplace | tokens, component specs |
| interaction-design | designer-skills marketplace | states, flows, micro-delight |
| visual-critique | designer-skills marketplace | audit mode |
| senior-product-designer | bundled here | the conductor |

See `guide.html` for the full explanation, the live-build script, and the per-company process.
